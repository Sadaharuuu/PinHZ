# PinHZ

![image](https://img.shields.io/badge/License-MIT-yellow.svg)
![image](https://img.shields.io/badge/Platform-Win%2FLinux%2FmacOS-blue.svg)

一款基于QT5.9.1的**自定义协议可视化组帧**的串口 / 网口调试助手，专为嵌入式 / 工控领域研发，支持「组帧→拼帧→收发→解帧→日志→保存」全流程，可模拟上位机 / 下位机，大幅降低协议调试成本，加速项目开发。

## 🌟 核心优势

- 🎨 **可视化组帧**：通过 TableWidget + 按钮控件，无需手写十六进制码流，点点鼠标即可按自定义协议组帧；
- 💾 **帧格式保存**：组成的帧可保存为CSV，便于后续使用；
- 🎯 **解好帧**：可以即时或手动解析码流，也可以按帧格式保存；
- 🔌 **多通道收发**：支持串口（RS232/485）、网口（TCP/UDP），兼容真实硬件 / 虚拟设备；
- 📝 **全流程日志**：记录收发时间戳、码流内容、状态，支持筛选 / 导出 / 清空；
- 🚀 **轻量高效**：无冗余功能，专注协议调试，高频操作无卡顿。
